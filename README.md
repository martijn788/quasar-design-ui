## Install the package

yarn

```bash
yarn add quasar-design-ui
```

npm

```bash
npm install quasar-design-ui
```

### use a components

```vue.js
<template>
  <div>
    <hero-section />
  </div>
</template>

<script lang="ts">
import { HeroSection } from 'quasar-design-ui'

export default {
  name: 'PageIndex',
  components: { HeroSection }
}
</script>
```
